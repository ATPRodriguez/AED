### [Codigo](https://github.com/ATPRodriguez/AED/tree/main/Elementos-basicos-en-php/src/public/Ejercicio9)
```php
//Funcion principal
function mostrar() {
    for ($i = 10; $i >=1; $i--) {
            echo "<h1>$i</h1>";
    }
    echo "<h1>Fin!</h1>";
}
```
