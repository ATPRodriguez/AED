# Ejercicio 6 - Suma acumulada.

<div align="justify">

## Calcula la suma de los numeros del 1 al 50 usando un bucle.
   


### [Codigo](https://github.com/ATPRodriguez/AED/tree/main/Elementos-basicos-en-php/src/public/Ejercicio6)
```php
//Funcion principal
function calcular() {
    $resultado = 0;

    for ($i = 1; $i <= 50; $i++) {
        $resultado += $i;
    }

    return $resultado;
}
```
